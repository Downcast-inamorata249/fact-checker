## 🛠 How to Use fact-checker

The app is made to be simple. Here are the main steps to check information:

- **Check Text Claims:** Copy a news article, quote, or statement and paste it into the text box. The app will search for trusted sources to show if the claim is true, false, or unclear.

- **Verify Images:** Upload a photo to see if it has been altered or used before in a different context.

- **Examine Videos:** Load a video file or link to detect possible deepfake signs or edits.

- **Multi-language Support:** The software supports all major languages. It can verify content in English, Spanish, French, Chinese, Arabic, and many others without needing changes or plugins.
